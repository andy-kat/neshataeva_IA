Training Planner

Простое приложение для учёта ваших тренировок.

Возможности
- Добавляйте тренировки: укажите дату, тип (например, кардио или йога) и длительность.
- Фильтруйте список: смотрите только нужные тренировки по типу или дате.
- Сохраняйте прогресс:все данные хранятся в файле и не теряются.

Как пользоваться
1. Запустите файл 
2. Введите дату (ДД.ММ.ГГГГ), выберите тип и укажите длительность.
3. Нажмите «Добавить тренировку».
4. Для поиска используйте фильтр по типу и дате.
5. Кнопка «Сбросить фильтр» покажет все тренировки.

Технические детали
- Приложение написано на Python.
- Для работы не нужны дополнительные настройки.
